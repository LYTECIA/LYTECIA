## 📂 Featured Project

### MentorMatch (in progress)

Concept of a web platform designed to improve the matching process between volunteers and students in mentoring programs.

**Problem**

The matching process is often done manually, which can lead to delays and less optimal mentor–student pairs.

This project explores how a structured recommendation system could support this process while keeping human validation.

**Main ideas**

- Compatibility scoring algorithm based on multiple criteria (location, interests, availability, etc.)
- Recommendation system to suggest the most compatible volunteer profiles
- Relational database design to manage volunteers and students data

**Tech stack**

PHP • MySQL • JavaScript
